kotlin重点语法

kotlin重点语法介绍

重点语法

kotlin中类名不用和文件名一致，包名不用和文件夹路径名一致

val 常量，相当于java的final
var 变量

val m = intArrayOf(1,2,3)
m是常量不能再赋值，但是可以改变数组里的值，如将第0个元素设置为4
m.set(0,4)
for(item in m){
println(item)
}

变量赋值要求类型必须一致，不能将小范围变量赋值给大范围的变量，kotlin中所有类型都是对象，可以把类型转换一致后赋值，如：
var x = 1
var y: Byte = 2
x = y.toInt()

if语句最后一个表达式可作为返回结果
var max = if(x > y){
println(“x > y”)
x
}

返回类型Int?表示返回值可能为Int也可能为null,如果不加?则返回值必须是Int，否则编译器报错
fun convert2Int(str: String): Int? {

}

sout + tab键在kotlin中是println()

“”” hello \n world “”” 三个双引号之间的内容会原封不动，也不会转义

循环中，it表示当前遍历的元素

Any是kotlin中的顶层类，相当于java中的Object

if(str is String) is相当于instanceOf

if(a !in 2..10) 表示a不在2到10之间，步进为1，本质2.rangeTo(10)，但是用..更简洁

step 2表示步进为2，downTo表示10到2之间

kotlin数组

Int类型编译成字节码在jvm中相当于原生类型int，IntArray相当于int[]

when关键字

类似于java中的if else

when{
x > y -> println(“x > y”)
}

更进一步，如果返回值是表达式可以用赋值
